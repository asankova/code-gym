# code-gym 🏋️

As an engineering manager, it's easy to let hands-on coding skills rust while you're busy with 1:1s, roadmaps, and strategy. 
This repo is my way of staying sharp - a single place for all ad hoc coding exercises, no matter the platform or language.
The goal isn't perfection. It's consistency and reps.

## Platforms

- **ProgrammingExpert** [programmingexpert.io](https://www.programmingexpert.io/product)
- **LeetCode** [asankova](https://leetcode.com/asankova)

## Notes
All Python solutions include type annotations, docstrings, and test cases. Each file is self-contained and can be run directly:

```bash
chmod +x solution.py
./solution.py
```
